## 概述

可控制正反转，配有兼容乐高的安装孔，兼容舵机，舵盘。搭配优质螺旋浆，可轻松吹灭20cm处的打火

机火焰，可用于救火机器人的制作，机器人设计开发必备。

## 模块参数

1.供电电压:5V
2.连接方式:3PIN防反接杜邦线
3.模块尺寸:4*2.1cm
4.安装方式:M4螺钉兼容乐高插孔固定

| 引脚名称 | 描述       |
| -------- | ---------- |
| V        | 5V电源引脚 |
| G        | GND 地线   |
| S        | 信号引脚   |
